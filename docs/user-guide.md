# User Guide — ReturnFlow

## For Customers

### How to Submit a Return Request

1. Visit the ReturnFlow service portal
2. Click Submit a Return Request from the catalog
3. Fill in the following fields:
   - Order Number
   - Last Name
   - Email Address
   - Reason for Return
4. If your reason is Other, provide details in the 
   Additional Comments field that appears
5. Click Submit

### What happens next

- You will receive an email immediately confirming your 
  return with a unique reference number and drop off 
  instructions
- Drop your item off at the specified collection point
- Once the item has been inspected you will receive an 
  email with the outcome
- If approved, your refund will be processed and you 
  will be notified
- If rejected, you will receive an email explaining 
  the reason for rejection

### Return Window
Returns must be submitted within 14 days of your order 
date. Requests outside this window will not be accepted.

### Need help?
Visit the Knowledge Base on the portal for a step by 
step guide on how to return an item.

## For Warehouse Fulfillers

### How to process a return task

1. You will receive a notification when a return task 
   is assigned to you
2. Log in to ServiceNow and open your assigned task
3. Click the Order Return reference field to view the 
   full return record and customer context
4. Physically inspect the returned item
5. Set the Assigned To field to yourself
6. Update the Task Outcome field:
   - Warehouse Approved: item meets return criteria
   - Warehouse Rejected: item does not meet criteria
7. If selecting Warehouse Rejected, enter the reason 
   in the Reason for Rejection field. This will be 
   automatically included in the email sent to the 
   customer
8. Set the task State to Closed
9. The system handles all next steps automatically

---

## For Finance Fulfillers

### How to process a refund task

1. You will receive a notification when a refund task 
   is assigned to you following warehouse approval
2. Log in to ServiceNow and open your assigned task
3. Review the return details via the Order Return 
   reference field
4. Process the refund on your payment system
5. Set the Assigned To field to yourself
6. Set Task Outcome to Finance Approved
7. Set the task State to Closed
8. The customer will automatically receive a refund 
   confirmation notification

---

## For Managers

### How to use the dashboard

1. Log in to ServiceNow
2. Navigate to the ReturnFlow dashboard
3. View reports covering:
   - Return volumes over time
   - Approval and rejection rates
   - Reasons for return
4. All reports update automatically as records 
   are processed. No manual input required.
